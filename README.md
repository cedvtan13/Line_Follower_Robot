# Line Follower Robot
This repository contains the firmware and designs for a series of high-performance line-following robots.

## Project Overview
The current version focuses on high-speed navigation using an STM32F411 microcontroller and a vacuum downforce system (EDF) for superior cornering stability.

## Robot Versions
*   **LineFollowerV3.1**: (Current) Features PID control, 8 IR sensors, and brushless motor downforce.
